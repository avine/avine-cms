
	=======================================================
	About the '/tmpl' directory of the newsletter component
	=======================================================


- The directory '/components/com_newsletter/tmpl' should contain one sub-directory for each newsletter template (like '/default').

- The others directories (like '/my_template') should contain only the images of each template.
But, the XHTML code of each template will be stored in the database (not in FTP file).

- The exception is the for the '/default' directory :
The XHTML code of the default template can be found in the 'tmpl.html' file.
It's allow you to load this template as an example, when you create a new template in the administration.
